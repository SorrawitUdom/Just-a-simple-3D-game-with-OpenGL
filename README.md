# Just a simple 3D Game with OpenGL

ซอร์สโค้ดนี้ถูกสร้างข้นโดย **บุคคลที่คุณก็รู้ว่าใคร** เพราะฉะนั้น ด้านในมันจะระเบิดเล็กน้อยนะครับ 

### Idea
เกมเกมนี้ถูกสร้างขึ้นมาในรุปแบบเกมขับรถแบบมุมมอง topview โดยได้รับแรงบันดาลใจมาจาก [Grand Theft Auto(1997)](https://en.wikipedia.org/wiki/Grand_Theft_Auto_(video_game))  (ซึ่งจริงๆแล้วนั้น GTA1 ดูดีกว่าเกมของผมมากๆ ดังนั้นผมจะเรียกเกมนี้อีกชื่อว่า GTA1-Demake)
